# SMKCoding_Kanesa_Yudas

TechPedia merupakan platform belajar online khususnya di bidang IT yang dikembangkan khusus untuk membantu kamu belajar agar lebih mudah dan terarah

# Used Features
* Retrofit
* OkHttp
* Picasso
* Shimmer
* SQLite Database
* Butterknife
* CircleImageView
* GifDrawable
